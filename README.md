VotingSystem

VotingSystem is a Java Swing application that simulates a simple online voting system. Users can vote for Generation Z, Millennials, or Generation Alpha and view the results in real time.

Key Features

User-Friendly GUI: A clean graphical interface built with Java Swing.

Real-Time Voting: Votes are counted immediately and results are displayed dynamically.

Input Validation: Ensures users select a voting option before submitting.

Event Handling: Demonstrates basic event-driven programming in Java.

State Management: Tracks votes internally for each generation.

Purpose

This project is ideal for learning and demonstrating:

Java Swing for desktop application development

Event-driven programming concepts

Managing and updating application state dynamically

Building simple, interactive GUI applications

Ideal For

Students and beginners learning Java GUI programming

Developers exploring event-driven design

Anyone interested in building simple interactive desktop applications
